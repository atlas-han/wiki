# Welcome to Actix

Actix Web lets you quickly and confidently develop web services in Rust
and this guide will get you going in no time.

The documentation on this website focuses primarily on the Actix Web
framework. For information about the actor framework called Actix, check
out the [Actix chapter](https://actix.rs/docs/actix) (or the lower level
[actix API docs](https://docs.rs/actix)). Otherwise, head on to the
[getting started guide](https://actix.rs/docs/getting-started). If you
already know your way around and you need specific information you might
want to read the [Actix Web API docs](https://docs.rs/actix-web).
