# Raw Book Cleaner

이 스킬은 `01.raw/books/` 디렉토리의 책 클리핑 파일에서 쇼핑몰 노이즈를 제거하고, 독서 노트로 활용 가능한 형태로 재구성합니다.

**권한**: 이 스킬은 `01.raw/books/` 디렉토리의 파일을 직접 수정할 권한이 사전에 부여되어 있습니다. 파일 수정 전에 별도 확인을 요청할 필요 없이 바로 편집하세요.

---

## 작업 흐름

### 1. 대상 파일 확인

사용자가 특정 파일을 지정하면 그 파일을, 지정하지 않으면 `01.raw/books/` 디렉토리의 모든 `.md` 파일을 대상으로 합니다.

```
/path/to/LLM-WIKI/01.raw/books/
```

### 2. 파일 읽기

파일 전체를 읽습니다. YAML frontmatter와 각 `## 섹션`을 파악합니다.

### 3. 추출할 콘텐츠 식별

아래 규칙에 따라 **보존할 내용**과 **제거할 내용**을 결정합니다.

---

## 보존 규칙

### YAML Frontmatter
- **그대로 유지** — 변경하지 않습니다.
- frontmatter 내 `isbn:`, `genre:`, `publisher:` 등 빈 필드도 그대로 둡니다.

### 기본정보 섹션 (`## 기본정보`)
- 표지 이미지 한 줄만 유지: `![cover](URL)`
- 파일에 `## 기본정보`가 여러 개 있으면 **첫 번째**(커버 이미지)만 유지
- HTML 테이블로 된 기본정보(ISBN, 쪽수, 크기 등)는 **제거**

### 작가정보 섹션 (`## 작가정보`)
- **주 저자(저자(글))의 약력 텍스트만 유지**
  - `### 저자(글) 저자명` 제목은 유지
  - 저자 카테고리 링크 (`[현대소설가＞영미작가](URL)` 같은 한 줄 링크) **유지**
  - 저자 약력 본문 텍스트 유지
  - 저자의 다른 저서 이미지 목록/링크 (`![...]`, `[책제목](URL)`) **제거**
  - `- [모두보기](URL)` 제거
- **번역자(번역) 섹션 전체 제거** (`### 번역 번역자명` 이하 모두)

### 책 속으로 (`## 책 속으로`)
- **전체 유지** — 인용문과 쪽수 표시 그대로 유지

### 출판사 서평 (`## 출판사 서평`)
- 파일에 출판사 서평이 여러 개 있으면 **가장 긴(마지막) 버전**만 유지
- 상단에 있는 짧고 끊긴 버전("…" 로 끝나는 것)은 제거

---

## 제거 규칙

다음 섹션은 **완전히 제거**합니다:

| 섹션/패턴 | 이유 |
|-----------|------|
| `## 페이지 내용` | 네비게이션 링크 (회원가입, 로그인 등) |
| `## {책 제목}` (가격/평점 섹션) | 쇼핑 정보 (가격, 평점, 배송) |
| `#### 이 책의 이벤트` | 한시적 이벤트 정보 |
| `#### 패키지` | 묶음 상품 정보 |
| `#### 북카드` | 카드뉴스 이미지 목록 |
| `#### 키워드 Pick` | AI 추천 키워드 |
| `## 책 소개` | 카테고리 분류, 수상내역 링크 목록 |
| `## 북 트레일러` | 유튜브 동영상 |
| `이 책의 총서` | 시리즈 도서 목록 |
| `## 목차` | 챕터 번호와 쪽수만 있는 목차 |
| `Klover 리뷰` 이하 | 독자 리뷰, 평점 통계 |
| `문장수집` 이하 | 문장 수집 서비스 |
| `## 교환/반품/품절 안내` 이하 | 교환/반품 정책 |
| 마케팅 태그라인 | "과학적 사실로 엮인..." 같은 단발성 문구 |

---

## 출력 섹션 순서

정리된 파일은 반드시 이 순서로 구성합니다:

```
---
(YAML frontmatter)
---
## 기본정보

![cover](URL)


## 작가정보

### 저자(글) 저자명

저자 약력 텍스트

## 책 속으로

(인용문들)

## 출판사 서평

(출판사 서평 전문)
```

---

## 실행 방법

1. 대상 파일을 Read 도구로 읽습니다.
2. 위 규칙에 따라 보존/제거 대상을 결정합니다.
3. 정리된 내용을 조합합니다.
4. **Edit 도구로 파일 전체를 덮어씁니다** (파일을 직접 수정할 권한이 있음).
5. 완료 후 어떤 섹션이 제거됐는지 간단히 보고합니다.

### 다수 파일 처리

`01.raw/books/`의 모든 파일을 정리할 때는 파일 하나씩 순서대로 처리합니다. 각 파일 처리 후 간단한 완료 메시지를 출력합니다.

---

## 예시

**제거되는 것들**: 이벤트 배너, 배송 안내, 가격 정보, 패키지 상품, 북카드 이미지 리스트, 독자 리뷰, 반품 정책, 번역가 소개, 저자의 다른 저서 이미지

**남는 것들**: 표지 이미지, 주저자 약력 텍스트, 책 본문 인용, 출판사가 작성한 서평 전문
