# Notes

사람과 AI가 **함께 편집**하는 공간. `raw/`와 달리 AI가 수정할 수 있고, `wiki/`와 달리 사람도 직접 작성한다.

## 용도

- **초안(drafts)**: 정리 전 생각, 아이디어, 메모
- **스크래치패드**: 일시적인 분석, 비교, 계산
- **협업 문서**: 사람이 초안 작성 → AI가 보완·정제
