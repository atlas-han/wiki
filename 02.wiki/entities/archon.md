# Archon

[[tech-bridge-harness-engineering]] 영상에서 발표자(추정 Cole Medin)가 *"my open-source harness builder"* 로 소개한 **오픈소스 하네스 빌더**. [[ralph-wiggum-method|Ralph Loop]] 같은 다중 세션 오케스트레이션 하네스를 사용자의 프로세스·소프트웨어 개발 수명주기(SDLC)에 맞춰 커스텀 구축하도록 돕는다.

## 위키에서 알려진 사실

- 목적: *"agentic engineering을 시작하는 가장 쉬운 방법"* — 자신만의 하네스(Ralph Loop류)를 더 커스텀하게 구축.
- 무료·오픈소스로 소개됨. [[harness-engineering]]의 4계층(다중 세션 오케스트레이션)을 실전화하는 도구.

> 미해결: repo 정확한 경로·기능 범위는 영상 설명만으로는 미확인(링크는 추정). 관련 1차 소스 ingest 시 보완.
